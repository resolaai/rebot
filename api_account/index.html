API

アカウント一覧取得API

アカウントの一覧を取得します。 ログインされているユーザの権限によって取得できる内容が異なります。
スーパーユーザ：全アカウントの一覧を取得
管理者・一般：当該ユーザが所属する企業内のアカウント一覧を取得

■ エンドポイント 

/accounts

リクエスト

■ リクエストメソッド 

GET

■ リクエストパラメータ

なし

レスポンス

以下の項目を持つオブジェクトのリストを返します。

項目説明
idユーザID
cid企業ID
emailEメールアドレス
is_active当該ユーザが有効か無効化を表す
role当該ユーザのロールID
person_in_chargeユーザ名
profile_imageユーザのプロフィールイメージ
company_info以下の会社情報を持つオブジェクト
company_name会社名
is_rtchat_enabled有人対応の有無
app_num_limitアプリケーション作成可能数
no_show_menu表示しないメニューの一覧

アカウント作成API

Emailアドレス、ロールを指定してアカウントの作成を行います。
登録後、本登録依頼のメールを送信します。

■ エンドポイント 

/accounts/

リクエスト

■ リクエストメソッド 

POST

■ リクエストパラメータ

パラメータ必須説明
email○追加するアカウントのEmailアドレス
role○追加するアカウントのロールID
company_name新たな会社のアカウントとして作成する場合は、会社名を入力
会社名が無い場合は、ログインユーザと同じ会社のユーザとなる。（新たな会社を作成できるのはスーパーユーザアカウントのみです。）
is_rtchat_enabled0: 有人チャットを許可しない場合
1: 有人チャットを許可する場合
app_num_limit作成可能なアプリケーションの数を指定します。無制限の場合は0を指定します。
no_show_menu表示しないメニューをカンマ区切りで指定します。
例）state,test,log

レスポンス

なし（共通レスポンスのみ）

アカウント更新API

アカウントの詳細情報を更新します。

■ エンドポイント 

/accounts/{user_id}

リクエスト

■ リクエストメソッド 

POST

■ リクエストパラメータ

パラメータ必須説明
email○更新するアカウントのEメールアドレス
roleロールID
person_in_charge名前
profile_imageプロフィールイメージ
company_name会社名
（会社情報を更新できるのはスーパーユーザアカウントのみです。）
is_rtchat_enabled0: 有人チャットを許可しない場合
1: 有人チャットを許可する場合
（会社情報を更新できるのはスーパーユーザアカウントのみです。）
app_num_limit作成可能なアプリケーションの数を指定します。無制限の場合は0を指定します。
（会社情報を更新できるのはスーパーユーザアカウントのみです。）
no_show_menu表示しないメニューをカンマ区切りで指定します。
例）state,test,log
（会社情報を更新できるのはスーパーユーザアカウントのみです。）

レスポンス

なし（共通レスポンスのみ）
