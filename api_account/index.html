API

アカウント作成API

Emailアドレス、ロールを指定してアカウントの作成を行います。
登録後、本登録依頼のメールを送信します。

■ エンドポイント 

/account/register

リクエスト

■ リクエストメソッド 

POST

■ リクエストパラメータ

パラメータ必須説明
email○追加するアカウントのEmailアドレス
role○追加するアカウントの役割
0: スーパーユーザ
1: 管理者
2: 一般
company_name新たな会社のアカウントとして作成する場合は、会社名を入力
会社名が無い場合は、ログインユーザと同じ会社のユーザとなる
is_rtchat_enabled0: 有人チャットを許可しない場合
1: 有人チャットを許可する場合

レスポンス

なし
