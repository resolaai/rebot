API

アプリケーション一覧取得API

アプリケーションの一覧を取得します

■ エンドポイント 

/apps

リクエスト

■ リクエストメソッド 

GET

■ リクエストパラメータ

なし

レスポンス

以下の項目を持つオブジェクトのリスト

項目説明
appアプリケーション名
descriptionアプリケーションの説明
typeアプリケーションタイプ
WebApp: Web API
LineApp: LINE API
FacebookApp: Facebook API
unique_idアプリケーションを識別するID
アプリケーション配下のAPI（アクション一覧など）をコールする際に利用する

アプリケーション詳細取得API

アプリケーションの詳細を取得します

■ エンドポイント 

/apps/{unique_id}

リクエスト

■ リクエストメソッド 

GET

■ リクエストパラメータ

なし

レスポンス

以下の項目を持つオブジェクト

項目説明
appアプリケーション名
descriptionアプリケーションの説明
typeアプリケーションタイプ
WebApp: Web API
LineApp: LINE API
FacebookApp: Facebook API
unique_idアプリケーションを識別するID
アプリケーション配下のAPI（アクション一覧など）をコールする際に利用する
parent_id親アプリケーションのunique_id
call_back_urlチャットボットのコールバックURL

以下の項目はタイプが"WebApp"の場合で、チャットウィンドウを設置する場合のみ

項目説明
header_colorチャットウィンドウのヘッダーカラー
例）#2baab1
header_titleチャットウィンドウのヘッダータイトル
user_balloon_bg_colorユーザ側の吹き出し背景色
character_imageチャットウィンドウのキャラクターイメージ名
allowed_domainsチャットウィンドウの設置を許可するドメイン名
複数ある場合はカンマ区切りで指定
ドメインを制限しない場合は"*"と指定
send_urlチャットウィンドウが送信する先のURL
指定しない場合はデフォルト送信先
allowd_ipsアクセス可能なIPアドレス

以下の項目はタイプが"LineApp"の場合のみ

項目説明
channel_idChannel ID
channel_secretChannel Secret
access_tokenAccess Token

以下の項目はタイプが"FacebookApp"の場合のみ

項目説明
access_tokenアクセストークン
call_back_tokenFacebook Webhooks コールバック設定時の認証用トークン

アプリケーション登録API

以下のJSON形式のリストにてアプリケーション情報を受け取り、アプリケーションの登録または更新を行います。

■ エンドポイント 

/apps

リクエスト

■ リクエストメソッド 

POST

■ リクエストボディ

以下の値を持つJSONオブジェクトのリスト

パラメータ必須説明
update_type更新タイプ
1:新規追加（既にアクション名が存在する場合はスキップ）
2:未使用（アプリケーションでは利用なし）
3:置き換え（既にアプリケーション名が存在する場合は置き換え、存在しない場合は追加）
4:削除（アプリケーション名を指定して削除する）
*何も指定しない場合は"3:置き換え"となる
app○アプリケーション名
descriptionアプリケーションの説明
type○アプリケーションタイプ
WebApp: Web API
LineApp: LINE API
FacebookApp: Facebook API
parent_id親アプリケーションのunique_id
デフォルトは"0"（親アプリなし）

以下の項目はタイプが"WebApp"の場合のみ

項目必須説明
is_chat_window_enabled○0: チャットウィンドウを設置しない
1: チャットウィンドウを設置する
header_color△チャットウィンドウのヘッダーカラー
例）#2baab1
is_chat_window_enabled=1の時、必須
header_title△チャットウィンドウのヘッダータイトル
is_chat_window_enabled=1の時、必須
user_balloon_bg_color△ユーザ側の吹き出し背景色
is_chat_window_enabled=1の時、必須
character_image△チャットウィンドウのキャラクターイメージ名
is_chat_window_enabled=1の時、必須
allowed_domains△チャットウィンドウの設置を許可するドメイン名
複数ある場合はカンマ区切りで指定
ドメインを制限しない場合は"*"と指定
is_chat_window_enabled=1の時、必須
send_urlチャットウィンドウが送信する先のURL
指定しない場合はデフォルト送信先
allowd_ipsアクセス可能なIPアドレス

以下の項目はタイプが"LineApp"の場合のみ

項目必須説明
channel_idChannel ID
channel_secretChannel Secret
access_tokenAccess Token

以下の項目はタイプが"FacebookApp"の場合のみ

項目必須説明
access_tokenアクセストークン
call_back_tokenFacebook Webhooks コールバック設定時の認証用トークン

  {"update_type":"3", "app":"app2", "type":"LineApp", "channel_id":"xxx", "channel_secret":"xxx", "access_token":"xxx"}]},
  {"update_type":"4", "app":"app3"},
]
