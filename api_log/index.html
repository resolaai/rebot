API

ログ取得API

検索条件を指定して、{unique_id}で指定したアプリケーションのログデータの一覧を取得します。

■ エンドポイント 

/apps/{unique_id}/logs

リクエスト

■ リクエストメソッド 

GET

■ リクエストパラメータ

パラメータ 必須 説明
date_from 開始日付（yyyy-mm-dd)
値が無い場合は今日日付で検索
date_to 終了日付（yyyy-mm-dd)
値が無い場合は今日日付で検索
action アクション名を前方一致で検索
state ステート名を前方一致で検索

■ 例 

https://rebot.jp/api/apps/xxxxxxxxxxxxxx/logs?date_from=2017-01-01&date_to=2017-03-31&action=sys

レスポンス

以下の情報を含むログ情報をリストで返します。

項目説明
userユーザを一意に識別するID
textユーザの会話
datetime日時
action認識したアクション名
stateステート名
response応答内容
result_jsonシステムで解析した情報
詳細はログ詳細画面を確認。
